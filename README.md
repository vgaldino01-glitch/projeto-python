# 🧮 Calculadora em Python

Projeto de uma calculadora simples desenvolvida em Python.

## 📌 Sobre o projeto

Esta aplicação realiza operações matemáticas básicas através da entrada de dados do usuário.

O projeto foi desenvolvido para praticar lógica de programação e os fundamentos da linguagem Python.

## 🚀 Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Interação com o usuário pelo terminal

## 🛠️ Tecnologias utilizadas

- Python
- VS Code
- Git
- GitHub

## 📚 Conceitos praticados

- Variáveis
- Strings
- Tipos de dados
- Entrada e saída de informações
- Estruturas condicionais
- Operações matemáticas

## ▶️ Como executar

No terminal, execute:

python3 calculadora.py
